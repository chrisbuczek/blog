# LAOTs State Management

Recently I saw Anton Mikhailov in [Wookash Podcast](https://www.youtube.com/watch?v=ShSGHb65f3M) talking about LAOTs - large arrays of things. It's an interesting state management design which makes code more performant and easier to read than OOP design pattern. It was used to create a game called [Dreams](https://www.playstation.com/pl-pl/games/dreams/) realeased on PlayStation 4 in 2020.
